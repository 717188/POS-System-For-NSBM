POS System – NSBM Group Project

Introduction

This is a Point of Sale (POS) system developed as a group project for SE204.3 – Development of Enterprise Applications.
The system helps manage sales, customers, products, suppliers, employees, and invoices in a simple and efficient way.


Features
Sales management
Customer management
Product management
Supplier management
Employee management
Invoice generation
Basic reporting system

Technologies Used

Java
MySQL
Java Swing (GUI)

NetBeans IDE

System Modules
Customer Module
Product Module
Supplier Module
Sales Module
Employee Module
Invoice Module

Requirements

Windows OS
Java JDK
MySQL Server
NetBeans IDE

How to Run

Import project into NetBeans
Start MySQL server
Import database file
Run the main Java file

Limitations

Works only on desktop
Requires database connection
Basic reporting only
Limited UI design

Conclusion

This POS system helps to improve business operations by managing sales and inventory efficiently. It reduces manual work and increases accuracy.
